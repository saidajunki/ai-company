# ダジャレPRリポジトリ 〜面白いダジャレを世界に届けよう〜

このリポジトリは「みんなでダジャレを投稿して楽しむ」OSSです。  
あなたの考えた最高のダジャレを、Pull Requestでぜひ投稿してください！

---

## 使い方・貢献方法

1. `dajare-list.txt` を1行追加してPR
    - 例:  
      ```
      寿司が好きすぎて、スシんどい。
      ```
2. PRには「ダジャレ採点コメント」や「思い出・こだわりポイント」などトークも大歓迎！

### 投稿ルール & ガイドライン

- 1PRにつき1ダジャレ推奨（複数でもOK）
- 著作権侵害・公序良俗違反・NGワードはNG（自動チェック有）
- 日本語で書いてネ（自動判定あり）
- 面白さ優先！ツッコミどころや解説も歓迎です
- インスパイア例:  
  ```
  パン屋でバターンキュウ。
  ```

### お手本PRサンプル

```
追加行:
焼き肉屋で網にやけにあみだくじ。

PRコメント:
「網」と「あみだくじ」で2重ダジャレにトライしました。焼けましたか？
```

---

## GitHub Actionsによる自動チェック

PR時、下記を自動チェックします：
- 新規ダジャレの日本語性/重複
- 長さ（40文字以内）
- NG語句フィルタ
- コミットが`dajare-list.txt`のみ変更か

## 概要
CLIで遊べる「擬音バトル復活祭」ツール。自動擬音語生成＋実況バトル＋履歴保存＋スコア集計に対応。

## 使い方

### バトルを実行
```
python3 src/onomatopoeia_battle.py --battle 太郎 次郎
```

### スコアを見る
```
python3 src/onomatopoeia_battle.py --score
```
