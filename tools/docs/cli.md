# Common CLI Tools (Shared)

These are **external** CLI tools that are useful to avoid rebuilding long shell commands.

## ripgrep (`rg`)

- Purpose: fast code/file search (preferred over `grep -R`).
- Check:

```bash
rg --version
```

- Install (Debian/Ubuntu):

```bash
apt-get update && apt-get install -y ripgrep
```

## jq

- Purpose: JSON processing.
- Check:

```bash
jq --version
```
